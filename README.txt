INSTRUKCJA URUCHOMIENIA

1. Zainstaluj Node.js (zalecane LTS).
2. W folderze projektu uruchom:
   npm install
   npm run dev

3. Otwórz lokalny adres (np. http://localhost:5173).

DEPLOY:
- Build komendą: npm run build
- Wynik pojawi się w folderze dist/
- Wgraj zawartość dist/ na dowolny hosting statyczny (np. Vercel, Netlify, cPanel, OVH).
